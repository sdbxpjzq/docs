# chrome常用快捷键

- `cmd+L` — 选中地址栏
- `cmd+option+i` — 调出开发者工具
- `cmd+option+j` == 调出js控制台(与上边的差不多)
- `cmd+shift+m` — 切换手机展示模式
- `cmd+option+u` == 查看页面源代码
- `cmd+option+左右方向键` == 切换页签
- `shift+[` == 上一页
- `shift+]` == 下一页
- `cmd+e` == 将选中的内容放入搜索框中
